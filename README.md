<h3>Installation</h3>

<h4>Maven</h4>

```html
	<repositories> 
		<repository>
		    <id>jitpack.io</id>
		    <url>https://jitpack.io</url>
		</repository>
	</repositories>
```

```html
	<dependency>
	    <groupId>com.github.duitkupg</groupId>
	    <artifactId>DuitkuSDK</artifactId>
	    <version>1.0</version>
	</dependency>
```

<h4>Build.gradle(Module:app)</h4>

```html
	implementation 'com.duitku:sdk-android:1.0'
```
